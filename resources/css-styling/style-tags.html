Style Tags vs External Stylesheets

There are two main ways to add CSS to your HTML: using <style> tags inside your HTML file (internal styling) or linking to an external CSS file.

Internal Styling Example

This text is styled using internal CSS defined within the <style> tags in the <head> section.

External Stylesheet Example

External stylesheets allow you to define styles in a separate CSS file and link to it using the <link> tag: 

<link rel="stylesheet" href="styles.css" />

From this point onward, we'll be using the external stylesheet, styles.css, to apply styles across all CSS example pages.
