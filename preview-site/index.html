Blueprint Website Workshop - Make Your Own Portfolio!

Welcome to Blueprint Workshop 1.0

This workshop is designed for participants with little to no coding experience. We will split it into two parts:

  1. Creating your Website
  2. Putting your Website on the Internet

If you're curious about anything here, feel free to check out Workshop Resources to find out more about HTML/CSS and websites.

Creating your Website

Start with the basics of HTML and CSS to build your first webpage. The focus will be on understanding HTML structure and styling elements using CSS.

Putting your Website on the Internet

Once your website is complete, you'll learn how to host it using GitHub Pages. This will involve creating a repository, uploading your files, and enabling GitHub Pages for your project to make your site accessible to anyone online.
